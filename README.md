# 🚀 BluPrint - Solana Meme Coin Launchpad

Launch your own Solana meme coin in seconds. No code. No friction. Just launch.

## ✨ Features

- ⚡ **Instant Token Creation** – Create SPL tokens in seconds (0.15 SOL)
- 🔒 **Secure Token** – Revoke Mint, Freeze & Update authorities (FREE)
- 🎁 **Referral System** – Earn 0.05 SOL per friend
- 📱 **Multi-Wallet** – Phantom, Solflare, Trust Wallet
- 🌍 **Multi-Language** – English, Русский, 中文
- 📊 **Admin Dashboard** – Track earnings, users, tokens
- 🔥 **Live Token List** – Watch tokens go live with 🔥 effect

## 🛠️ Tech Stack

- **Frontend**: Next.js 16, TypeScript, Tailwind CSS, Framer Motion
- **Backend**: Next.js API Routes, Upstash Redis
- **Blockchain**: Solana Web3, SPL Token, Metaplex
- **Deployment**: Vercel
